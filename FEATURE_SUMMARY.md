# 궁합 분석 관계 타입 기능

## 구현 날짜

2026-02-12

## 추가된 기능

### 1. 관계 선택 UI

- **위치**: `/protected/analysis/compatibility`
- **컴포넌트**: 관계 선택 드롭다운 (Select)
- **특징**:
  - 5개 카테고리로 그룹화
  - 각 관계마다 이모지 + 설명
  - 선택 시 관계 설명 자동 표시

### 2. 관계별 맞춤 분석

#### 연인 관계

- **애인**: 연애 궁합, 결혼 가능성
- **소개팅/만남**: 첫인상, 발전 가능성
- **짝사랑**: 상대방 마음, 다가가는 타이밍
- **결혼 예정**: 결혼 생활 조화, 장기 안정성

#### 가족 관계

- **부부**: 갈등 해소, 장기 관계 유지
- **부모-자녀**: 세대 차이 극복, 효과적 소통
- **형제/자매**: 경쟁보다 협력, 평생 우애
- **시댁/처가**: 상호 존중, 갈등 예방

#### 친구 관계

- **친구**: 우정 지속성, 함께 성장
- **절친**: 깊은 신뢰, 평생 우정
- **룸메이트**: 생활 습관 조화, 편안한 동거

#### 직장 관계

- **상사-부하**: 지시-수행 궁합, 성과 극대화
- **동료**: 업무 협업 시너지, 팀워크
- **멘토-멘티**: 교육-학습 궁합, 성장 가능성
- **알바-사장**: 단기 협력 효율

#### 비즈니스

- **동업자**: 비즈니스 비전 일치, 역할 분담
- **투자자-창업자**: 신뢰 형성, 사업 성공 시너지
- **고객-공급자**: 거래 지속성, 장기 관계
- **프로젝트 팀원**: 단기 목표 달성, 스트레스 관리

## 기술 구현

### 파일 변경사항

#### 신규 파일 (1개)

```
lib/constants/relationship-types.ts - 관계 타입 상수 정의
```

#### 수정 파일 (2개)

```
app/protected/analysis/compatibility/compatibility-client.tsx
  - 관계 선택 드롭다운 추가
  - relationship state 추가
  - analyzeCompatibilityAction에 relationship 전달

app/actions/compatibility-analysis-action.ts
  - relationship 파라미터 추가
  - getRelationshipGuide() 함수 추가
  - AI 프롬프트에 관계별 가이드라인 포함
```

### AI 프롬프트 구조

```typescript
// 기존 프롬프트에 추가된 섹션
## 두 사람의 관계
${relationshipGuide}  // 관계별 분석 포커스 설명

// 예시: 동업자 관계
**동업자 관계** - 사업 파트너로서의 시너지를 분석합니다.
- 비즈니스 비전 일치도
- 역할 분담 최적화
- 재무 관리 조화
- 장기 파트너십 전략
```

## 사용 흐름

```
1. /protected/analysis/compatibility 접속
   ↓
2. 첫 번째 사람 선택
   ↓
3. 두 번째 사람 선택
   ↓
4. 관계 타입 선택 (드롭다운)
   ↓
5. "궁합 분석하기" 클릭
   ↓
6. AI가 관계 타입에 맞춰 분석 수행
   ↓
7. 결과 표시 (관계별 맞춤 조언 포함)
```

## 예시

### 애인 관계 분석

- 강점: "서로 다른 매력으로 끌림", "감정 표현 방식이 조화로움"
- 주의: "금전 관리 의견 차이", "미래 계획 조율 필요"
- 조언: "연애는 두 사람이 함께 그리는 그림입니다. 서로의 색깔을 존중하며..."

### 동업자 관계 분석

- 강점: "비즈니스 감각 시너지", "역할 분담이 명확함"
- 주의: "의사결정 방식 차이", "리스크 감수 성향 조율"
- 조언: "사업은 신뢰가 기반입니다. 계약서보다 중요한 것은 서로에 대한 믿음..."
