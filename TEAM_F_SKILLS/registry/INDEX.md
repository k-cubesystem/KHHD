# 스킬 인덱스 (TEAM_F)

> 이 폴더에는 재사용 가능한 스킬 구현 파일이 위치합니다.
> 각 스킬은 독립적으로 실행 가능해야 합니다.

---

## 파일 명명 규칙

```
SKILL-{번호}-{kebab-case-이름}.md
예: SKILL-001-supabase-query-builder.md
    SKILL-002-gemini-streaming-response.md
```

## 스킬 파일 템플릿

```markdown
# SKILL-{번호}: {스킬명}

## 목적

이 스킬이 해결하는 문제

## 사용법

입력 파라미터와 호출 방법

## 구현

코드 또는 단계별 절차

## 주의사항

알려진 제한 또는 주의 사항

## 테스트

정상 동작 확인 방법
```
