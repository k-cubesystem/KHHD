# 사용자 가이드

해화당(Haehwadang) 사주 분석 서비스 사용 가이드입니다.

---

## 시작하기

### 회원가입

1. 홈페이지에서 **"시작하기"** 버튼을 클릭합니다.
2. **카카오/구글/이메일** 중 원하는 방법으로 가입합니다.
3. 프로필 정보를 입력합니다:
   - 이름
   - 생년월일 (양력/음력)
   - 태어난 시간 (선택)
   - 성별

### 로그인

- 가입 시 사용한 소셜 계정 또는 이메일로 로그인합니다.
- 로그인 상태는 30일간 유지됩니다.

## 부적 시스템

### 부적이란?

**부적(符籍)**은 해화당에서 사용하는 서비스 이용권입니다. 각 기능을 이용할 때 부적이 차감됩니다.

### 기능별 부적 비용

| 기능 | 부적 | 설명 |
|------|------|------|
| 기본 사주 분석 | 1개 | 사주팔자 기본 분석 |
| 관상 분석 | 2개 | AI 얼굴 관상 분석 |
| 손금 분석 | 2개 | AI 손금 분석 |
| 풍수 분석 | 2개 | 인테리어 풍수 분석 |
| 이미지 생성 | 3개 | 개운 이미지 생성 |
| 오늘의 운세 | 무료* | *멤버십 회원 |

### 부적 충전 방법

1. **대시보드**에서 **"부적 충전"** 버튼을 클릭합니다.
2. 원하는 충전 금액을 선택합니다:

| 금액 | 부적 | 보너스 |
|------|------|--------|
| 3,000원 | 3개 | - |
| 5,000원 | 5개 | +1개 보너스 |
| 10,000원 | 10개 | +3개 보너스 |
| 30,000원 | 30개 | +10개 보너스 |

3. **Toss Payments**로 결제합니다.
4. 결제 완료 즉시 부적이 충전됩니다.

### 환불 정책

- 충전 후 7일 이내, 미사용 부적에 한해 환불 가능
- 사용된 부적은 환불 불가
- 보너스 부적은 환불 대상에서 제외

## 멤버십

### 해화 멤버십 혜택

월 **9,900원**으로 다음 혜택을 누리세요:

- 매월 **부적 10개** 지급 (3만원 상당)
- **오늘의 운세** 매일 무료 열람
- 분석 결과 **PDF 평생 보관**
- 카카오톡 **매일 운세 알림** (선택)
- 신규 기능 **우선 체험**

### 멤버십 가입 방법

1. **대시보드** → **"멤버십"** 메뉴로 이동합니다.
2. **"멤버십 시작하기"** 버튼을 클릭합니다.
3. 결제 정보를 입력합니다:
   - 카드 정보 (Toss Payments 보안 처리)
4. **"구독 시작"**을 클릭하면 즉시 활성화됩니다.

### 멤버십 관리

**메뉴**: 대시보드 → 멤버십 → 관리

- **다음 결제일** 확인
- **결제 수단 변경**
- **구독 해지**
  - 해지해도 현재 결제 기간 끝까지 이용 가능
  - 언제든 재가입 가능

### 자동 결제

- 매월 가입일에 자동 결제
- 결제 3일 전 카카오톡 알림
- 결제 실패 시 3일간 재시도 후 구독 일시정지

## 사주 분석

### 기본 사주 분석

1. **대시보드** → **"내 사주"** 또는 **"사주 분석"** 클릭
2. 분석할 대상을 선택합니다:
   - 본인
   - 등록된 가족
3. 생년월일시 정보가 맞는지 확인합니다.
4. **"분석 시작"** 버튼 클릭
5. AI가 분석 중... (약 10-20초)
6. 결과 확인:
   - 사주팔자 구성
   - 오행 분포
   - 성격 분석
   - 운세 흐름

### 결과 해석 가이드

#### 사주팔자 (四柱八字)

```
      년주      월주      일주      시주
천간   甲        丙        戊        庚
지지   子        寅        午        申
```

- **년주**: 조상운, 어린 시절
- **월주**: 부모운, 청년 시절
- **일주**: 본인, 배우자운
- **시주**: 자녀운, 노년 시절

#### 오행 (五行)

| 오행 | 색상 | 의미 |
|------|------|------|
| 木 | 청색 | 성장, 인내, 건강 |
| 火 | 적색 | 열정, 명예, 예절 |
| 土 | 황색 | 신뢰, 안정, 재물 |
| 金 | 백색 | 결단, 의리, 변화 |
| 水 | 흑색 | 지혜, 유연, 소통 |

#### 대운 (大運)

- 10년 단위로 변화하는 큰 운의 흐름
- 그래프에서 현재 대운 위치 확인 가능
- 점수가 높을수록 운이 좋은 시기

## 관상 분석

### 분석 방법

1. **대시보드** → **"관상 분석"** 클릭
2. **분석 목표**를 선택합니다:
   - **재물운**: 코, 이마 중심 분석
   - **도화운**: 눈, 입술 중심 분석
   - **관운**: 눈썹, 턱선 중심 분석
3. **얼굴 사진**을 업로드합니다:
   - 정면 사진 권장
   - 밝은 조명
   - 이목구비가 잘 보이는 사진
4. **"분석 시작"** 클릭
5. 결과 확인:
   - 오관(五官) 분석
   - 삼정(三停) 분석
   - 개선 조언
   - 추천 혈자리

### 분석 항목 설명

#### 오관 (五官)

- **귀**: 복덕, 수명
- **눈썹**: 의지, 형제운
- **눈**: 마음, 지혜
- **코**: 재물, 자존심
- **입**: 복록, 인간관계

#### 삼정 (三停)

- **상정 (이마)**: 지혜, 초년운 (0-30세)
- **중정 (눈~코)**: 의지, 중년운 (30-50세)
- **하정 (입~턱)**: 실행력, 말년운 (50세+)

## 손금 분석

### 분석 방법

1. **대시보드** → **"손금 분석"** 클릭
2. **손바닥 사진**을 업로드합니다:
   - 손바닥 전체가 보이도록
   - 손금이 선명하게 보이는 사진
   - 오른손 권장 (활동적 운명)
3. **"분석 시작"** 클릭
4. 결과 확인:
   - 주요 선 분석
   - 건강 지표
   - 추천 혈자리

### 주요 손금

- **생명선**: 체력, 건강
- **두뇌선**: 사고방식, 지적 능력
- **감정선**: 감정, 인간관계
- **운명선**: 직업운, 성공

## 풍수 인테리어

### 분석 방법

1. **대시보드** → **"풍수 분석"** 클릭
2. **분석 테마**를 선택합니다:
   - **재물**: 돈이 모이는 공간
   - **연애**: 사랑이 피는 공간
   - **건강**: 활력 넘치는 공간
3. **방 종류**를 선택합니다 (거실, 침실, 서재 등)
4. **현재 방 사진**을 업로드합니다
5. **"분석 시작"** 클릭
6. 결과 확인:
   - 현재 상태 진단
   - 문제점 목록
   - 개선 쇼핑 리스트
   - 개선 후 예상 이미지

## 가족 관리

### 가족 추가

1. **대시보드** → **"가족 관리"** 클릭
2. **"가족 추가"** 버튼 클릭
3. 가족 정보 입력:
   - 이름
   - 관계 (부모, 배우자, 자녀 등)
   - 생년월일
   - 태어난 시간 (선택)
   - 성별
4. **"저장"** 클릭

### 가족 궁합 보기

1. **가족 관리** 페이지에서 두 명을 선택
2. 하단에 나타나는 **"궁합 보기"** 클릭
3. 궁합 매트릭스에서 점수 확인:
   - 90점 이상: 천생연분
   - 70-89점: 상생
   - 50-69점: 조화 필요
   - 50점 미만: 주의

## 자주 묻는 질문 (FAQ)

### Q: 태어난 시간을 모르면 분석이 안 되나요?

**A**: 시간 없이도 분석 가능합니다. 다만 시주(時柱) 정보가 빠지므로 분석 정확도가 약간 낮아질 수 있습니다.

### Q: 음력 생일은 어떻게 입력하나요?

**A**: 생년월일 입력 시 **"음력"** 옵션을 선택하시면 됩니다. 시스템에서 자동으로 양력 변환하여 계산합니다.

### Q: 분석 결과를 저장할 수 있나요?

**A**: 모든 분석 결과는 **"분석 기록"** 메뉴에서 확인 가능합니다. 멤버십 회원은 PDF로 다운로드할 수 있습니다.

### Q: 결제 영수증은 어디서 받나요?

**A**: **대시보드** → **"결제 내역"** 메뉴에서 각 결제 건별로 영수증을 확인하고 다운로드할 수 있습니다.

### Q: 구독 해지 후 재가입하면 혜택이 유지되나요?

**A**: 재가입 시 신규 가입과 동일한 조건으로 시작됩니다. 이전 분석 기록은 그대로 유지됩니다.

### Q: AI 분석의 정확도는 어느 정도인가요?

**A**: AI 분석은 전통 명리학 이론과 빅데이터를 기반으로 하지만, 재미와 참고용으로 활용해 주시기 바랍니다. 중요한 결정은 전문가와 상담하시는 것을 권장합니다.
