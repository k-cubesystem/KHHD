## 시작하기

### 회원가입

1. **홈페이지 접속**
   - https://haehwadang.com 접속
   - "시작하기" 버튼 클릭

2. **이메일 회원가입**
   - 이메일 주소 입력
   - 비밀번호 설정 (최소 6자)
   - 인증 이메일 확인

3. **초대 코드 (선택)**
   - 친구 초대 코드가 있다면 입력
   - 보너스 부적 3개 지급

### 첫 로그인

회원가입 후 자동으로 대시보드로 이동합니다.

**대시보드에서 확인할 수 있는 정보**:
- 보유 부적 개수
- 2026년 병오년 신년운세 배너
- 빠른 액세스 메뉴

## 멤버십 가입

### 멤버십 플랜

해화당은 3가지 멤버십 플랜을 제공합니다.

| 플랜 | 가격 | 혜택 |
|------|------|------|
| **무료** | 0원/월 | • 매일 부적 1개<br>• 가족 구성원 3명<br>• 분석 결과 7일 보관 |
| **프리미엄** | 9,900원/월 | • 매일 부적 5개<br>• 가족 구성원 10명<br>• 분석 결과 무제한 보관<br>• 우선 고객 지원 |
| **VIP** | 29,900원/월 | • 매일 부적 20개<br>• 가족 구성원 무제한<br>• 분석 결과 무제한 보관<br>• 1:1 전문 상담<br>• 프리미엄 AI 기능 |

### 멤버십 가입 절차

1. **플랜 선택**
   - 우측 상단 메뉴 → "멤버십" 클릭
   - 원하는 플랜의 "구독하기" 버튼 클릭

2. **결제 정보 입력**
   - Toss Payments를 통한 안전한 결제
   - 카드 정보 입력
   - 정기 결제 동의

3. **구독 완료**
   - 즉시 혜택 적용
   - 매일 부적 자동 충전
   - 이메일로 영수증 발송

### 멤버십 관리

**구독 확인**:
- 프로필 → "멤버십 관리"
- 현재 플랜 및 다음 결제일 확인

**플랜 변경**:
- 더 높은 플랜: 즉시 업그레이드
- 더 낮은 플랜: 다음 결제일부터 적용

**구독 취소**:
- "구독 취소" 버튼 클릭
- 다음 결제일까지 혜택 유지
- 언제든지 재구독 가능

## 부적(크레딧) 충전

### 부적이란?

부적은 해화당의 가상 화폐로, AI 분석 기능을 이용할 때 사용됩니다.

**소비량**:
- 사주 상세 분석: 1개
- 천지인 통합 분석: 1개
- 궁합 분석: 1개
- 관상/손금 분석: 각 1개
- 풍수지리 분석: 1개
- 오늘의 운세: 무료
- AI 역술인 대화: 무료

### 부적 충전 방법

#### 1. 직접 충전

**단계**:
1. 대시보드 우측 상단 부적 아이콘 클릭
2. "충전하기" 버튼 클릭
3. 충전 수량 선택:
   - 10개: 9,900원 (990원/개)
   - 50개: 39,900원 (798원/개) - 20% 할인
   - 100개: 69,900원 (699원/개) - 30% 할인

4. 결제 진행 (Toss Payments)
5. 즉시 지갑에 반영

#### 2. 멤버십 자동 충전

- **프리미엄**: 매일 자동으로 5개 충전
- **VIP**: 매일 자동으로 20개 충전

#### 3. 이벤트 및 보너스

- 친구 초대: 초대받은 친구 가입 시 3개
- 첫 결제: 보너스 5개
- 생일 축하: 생일 당일 10개

### 부적 내역 확인

**거래 내역**:
- 프로필 → "지갑" → "거래 내역"
- 충전/사용 내역 모두 확인 가능
- 최근 50건 표시

## 사주 등록

### 가족 구성원 추가

AI 분석을 받으려면 먼저 사주 정보를 등록해야 합니다.

**단계**:
1. **사주 관리 페이지 접속**
   - 메뉴 → "사주 관리" 클릭
   - 또는 분석 허브에서 "사주 관리" 카드 클릭

2. **구성원 추가**
   - "새 구성원 추가" 버튼 클릭
   - 정보 입력:
     - 이름: 실명 권장
     - 관계: 본인, 배우자, 부모, 자녀 등
     - 생년월일: YYYY-MM-DD 형식
     - 생시: HH:MM (모를 경우 "모름" 선택)
     - 양력/음력: 태어난 날짜 기준
     - 성별: 남성/여성

3. **저장**
   - "저장" 버튼 클릭
   - 즉시 분석 가능

### 중요 안내

**생시(生時)**:
- 정확한 생시는 사주 분석의 핵심입니다
- 모를 경우 "모름" 선택 가능
- 단, 시주(時柱) 없이 3주만으로 분석됩니다

**양력/음력**:
- 출생신고서 기준으로 선택
- 음력: 한글 달력 기준
- 양력: 서양 달력 기준
- 잘 모를 경우 부모님께 확인 권장

## 사주 분석 이용하기

### 1. 분석 허브

**접속 방법**:
- 메뉴 → "사주풀이"
- 대시보드 → "분석 허브" 카드

**4가지 분석 메뉴**:

#### ① 천지인 사주풀이
- 하늘(天), 땅(地), 사람(人) 통합 분석
- 사주 사주(四柱) 기본 분석
- 관상/손금 추가 가능 (선택)
- 거주지 풍수 보정 (선택)

**이용 방법**:
1. 카드 클릭
2. 프로필 선택 (본인 또는 가족)
3. 단계별 진행:
   - 1단계: 대상 선택
   - 2단계: 관상/손금 사진 업로드 (선택)
   - 3단계: 거주지 입력 (선택)
4. "분석 시작" 클릭
5. 10-20초 대기
6. 결과 페이지로 이동

#### ② 병오년 신년종합운세
- 2026년 붉은 말의 해 특화 분석
- 한 해 전체 운세 예측
- 월별 운세 흐름
- 주의사항 및 개운 방법

**이용 방법**:
1. 카드 클릭
2. 프로필 선택
3. "신년운세 확인하기" 클릭
4. AI 분석 결과 확인

#### ③ 오늘의 운세
- 매일 아침 업데이트
- 당일 종합 운세
- 재물운, 애정운, 건강운, 직업운
- 행운의 색상 및 숫자
- 조언 및 주의사항

**이용 방법**:
1. 카드 클릭
2. 프로필 선택
3. 즉시 오늘의 운세 확인
4. 무료 (부적 소비 없음)

#### ④ 사주 관리
- 가족 구성원 등록/수정/삭제
- 저장된 분석 결과 확인
- 궁합 매트릭스 (출시 예정)

---

### 2. 궁합 분석

**이용 방법**:
1. 메뉴 → "궁합"
2. "궁합 분석 시작" 클릭
3. 두 사람 선택:
   - 나 + 등록된 가족
   - 또는 초대 링크로 상대방 초대
4. 분석 시작 (부적 1개 소비)
5. 결과 확인:
   - 궁합 점수 (0-100점)
   - 강점 및 주의사항
   - 개선 조언

### 3. 관상/손금 분석

**관상 분석**:
1. 메뉴 → "관상/손금"
2. "관상 분석" 선택
3. 정면 얼굴 사진 업로드
   - 조명이 밝은 곳에서 촬영
   - 정면 응시
   - 표정은 자연스럽게
4. 분석 목표 선택: 재물운/애정운/직업운/건강운
5. 결과 확인

**손금 분석**:
1. "손금 분석" 선택
2. 손바닥 전체 사진 업로드
   - 손바닥을 쫙 펴서 촬영
   - 손금이 선명하게 보이도록
3. 분석 결과 확인

### 4. 풍수지리

**이용 방법**:
1. 메뉴 → "풍수지리"
2. 공간 사진 업로드 (거실, 침실, 사무실 등)
3. 분석 테마 선택: 재물운/건강운/애정운/직업운
4. 방 유형 입력
5. AI 풍수 분석 결과:
   - 현재 공간 평가
   - 개선 방안
   - 인테리어 조언

## 사주 용어 해석 가이드

### 기본 용어

#### 천간(天干) - 10개
**오행 속성**:
- 木(나무): 갑(甲), 을(乙)
- 火(불): 병(丙), 정(丁)
- 土(흙): 무(戊), 기(己)
- 金(쇠): 경(庚), 신(辛)
- 水(물): 임(壬), 계(癸)

**음양**:
- 양: 갑, 병, 무, 경, 임
- 음: 을, 정, 기, 신, 계

#### 지지(地支) - 12개
**띠(동물)**:
- 자(子): 쥐
- 축(丑): 소
- 인(寅): 호랑이
- 묘(卯): 토끼
- 진(辰): 용
- 사(巳): 뱀
- 오(午): 말
- 미(未): 양
- 신(申): 원숭이
- 유(酉): 닭
- 술(戌): 개
- 해(亥): 돼지

#### 사주 사주(四柱)

```
年柱 (연주) - 태어난 해
月柱 (월주) - 태어난 달
日柱 (일주) - 태어난 날
時柱 (시주) - 태어난 시간
```

**예시**:
```
홍길동 (1990년 1월 1일 14:30생)
年: 己巳 (기사) - 흙 뱀
月: 丙子 (병자) - 불 쥐
日: 甲寅 (갑인) - 나무 호랑이
時: 辛未 (신미) - 쇠 양
```

### 오행(五行) 해석

**木(나무)**:
- 성장, 확장, 창의성
- 봄, 청색, 동쪽
- 간, 담

**火(불)**:
- 열정, 활력, 밝음
- 여름, 적색, 남쪽
- 심장, 소장

**土(흙)**:
- 안정, 신뢰, 중재
- 환절기, 황색, 중앙
- 비장, 위장

**金(쇠)**:
- 결단, 정의, 견고함
- 가을, 백색, 서쪽
- 폐, 대장

**水(물)**:
- 지혜, 유연, 침착
- 겨울, 흑색, 북쪽
- 신장, 방광

### 오행 균형

**건강한 사주**:
- 5가지 오행이 골고루 분포 (각 20%)

**편중된 사주**:
- 특정 오행이 과다하거나 부족
- 개운 방법으로 보완 필요

**예시**:
```
木: 10% (부족) → 청색 옷, 동쪽 방향, 나무 소재
火: 40% (과다) → 물 마시기, 휴식, 파란색 사용
土: 20% (적정)
金: 15% (약간 부족)
水: 15% (약간 부족)
```

### 용신(用神)

**정의**: 사주의 균형을 맞춰주는 핵심 오행

**활용**:
- 용신에 해당하는 색상 옷 착용
- 용신 방향으로 집 배치
- 용신 오행 관련 직업 선택

## FAQ

### 일반

**Q: 회원가입은 무료인가요?**
A: 네, 회원가입 및 무료 플랜 이용은 완전 무료입니다.

**Q: 부적은 환불이 가능한가요?**
A: 미사용 부적에 한해 구매일로부터 7일 이내 100% 환불 가능합니다.

**Q: 멤버십은 언제든지 취소 가능한가요?**
A: 네, 언제든지 취소 가능하며 다음 결제일까지 혜택이 유지됩니다.

### 사주 정보

**Q: 생시를 모르는데 분석이 가능한가요?**
A: 네, 가능합니다. 다만 시주(時柱) 없이 3주로만 분석되어 정확도가 다소 떨어집니다.

**Q: 양력/음력을 잘못 입력했어요.**
A: 사주 관리 페이지에서 수정 가능합니다. 수정 후 다시 분석하시면 됩니다.

**Q: 가족 정보를 삭제하면 분석 결과도 삭제되나요?**
A: 네, 구성원 삭제 시 관련된 모든 분석 결과도 함께 삭제됩니다.

### 분석 결과

**Q: 분석 결과는 얼마나 보관되나요?**
A: 무료 플랜은 7일, 프리미엄 이상은 무제한 보관됩니다.

**Q: 같은 날 분석해도 결과가 달라요.**
A: AI가 매번 새롭게 해석하므로 표현이 조금씩 다를 수 있으나, 핵심 내용은 동일합니다.

**Q: 오늘의 운세는 매일 바뀌나요?**
A: 네, 매일 자정(00:00)에 새로운 운세로 업데이트됩니다.

### 기술 지원

**Q: 결제가 안 돼요.**
A: Toss Payments 결제 화면에서 카드 정보를 다시 확인해주세요. 문제가 계속되면 고객센터(support@haehwadang.com)로 문의해주세요.

**Q: 분석이 너무 오래 걸려요.**
A: 일반적으로 10-20초 소요됩니다. 1분 이상 걸리면 새로고침 후 다시 시도해주세요.

**Q: 사진 업로드가 안 돼요.**
A: JPG, PNG 형식만 지원하며 파일 크기는 10MB 이하여야 합니다.
