# 해화당 AI 2.0 마스터 플랜 (MASTER_PLAN)

## 1. 서비스 비전
**"운명을 읽는 눈, 삶을 바꾸는 논리"**
전통의 직관과 현대의 데이터 사이언스가 결합된 인류 최고의 성공 가이드 서비스.

## 2. 핵심 가치 (Value Proposition)
- **High-End Accuracy**: KAIST 데이터 알고리즘을 통한 오행 분석의 정밀화.
- **Multimodal Insight**: 관상, 손금, 공간 풍수 사진 분석을 통한 입체적 운명 진단.
- **Actionable Prescription**: 단순한 길흉화복을 넘어, 오늘 당장 실천할 수 있는 '개운(開運)' 처방 제공.

## 3. 단계별 로드맵

### Phase 1: 기반 안정화 (Stabilization)
- **OAuth 완전 해결**: 구글/카카오 로그인 리다이렉트 및 세션 유지 이슈 해결.
- **Master Dashboard**: 사용자의 현재 운 에너지를 대시보드에서 즉시 확인.
- **기존 기능 폴리싱**: 사주, 관상, 손금 분석의 UI 연출 강화.

### Phase 2: 천지인(天地人) 통합 (Integration) - [신규]
- **지(地) - 풍수 분석**: 거주지/사무실 사진 분석을 통한 기운 배치 가이드.
- **통합 리포트**: 천(사주), 지(풍수), 인(관상/손금) 데이터가 결합된 5,000자 프리미엄 비록 완성.

### Phase 3: 관계 및 소셜 (Social & Relation) - [확장]
- **인맥 궁합 시스템**: 가족/지인 프로필 간의 호환성 점검.
- **카카오톡 알림톡**: 오늘의 운세 및 행운의 처방 자동 발송.

### Phase 4: 비즈니스 고도화 (Business) - [수익화]
- **Toss 결제 시스템**: 리포트 개별 판매 및 정기 구독 모델 적용.
- **운명 스토어**: 맞춤형 개운 아이템 추천.

## 4. 기술 스택 고도화
- **FE**: Next.js 14, Framer Motion (애니메이션), Tailwind CSS.
- **BE**: Supabase (Auth, DB, Storage, Edge Functions).
- **AI**: Gemini 1.5 Pro (Multimodal & LLM), Claude Code (FE 정밀 작업 협업).
