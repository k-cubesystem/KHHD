## 1. 프로젝트 비전

**"Premium 운명 공학 SaaS"**

전통 명리학의 통찰과 현대 AI의 논리를 결합. 타겟: 삶의 방향을 찾고자 하는 2040 현대인.

**핵심 가치**: 신뢰감(Trust) · 신비로움(Mystical) · 편안함(Comfort)

---

## 2. 핵심 기능

| 기능             | 설명                                          |
| ---------------- | --------------------------------------------- |
| **천지인 분석**  | 사주(天) + 관상/손금(人) + 풍수(地) 통합 분석 |
| **멀티 프로필**  | 본인 + 가족 최대 10명 관리                    |
| **오늘의 운세**  | 매일 업데이트 AI 운세                         |
| **궁합 분석**    | 2인 사주 기반 궁합 매칭                       |
| **AI 신당 채팅** | Gemini 기반 무제한 상담                       |
| **SNS 공유**     | 분석 결과 카드 생성 및 공유                   |

## 3. 수익 모델

### 복채 시스템 (기본 화폐)

- **1복채 = 1만원**
- 충전: 10만/30만/50만/100만원 패키지

### 멤버십

| 등급     | 일일 한도  | 가격     |
| -------- | ---------- | -------- |
| SINGLE   | 10만원/일  | 개인용   |
| FAMILY   | 30만원/일  | 가족용   |
| BUSINESS | 100만원/일 | 비즈니스 |

### 서비스 비용

| 서비스         | 복채 |
| -------------- | ---- |
| 테마 운세      | 1    |
| 관상/손금/풍수 | 2    |
| 천지인 사주    | 5    |

### 보너스 시스템

- 출석체크: 일 1복채 / 7일 연속 +3복채 보너스
- 룰렛: 1/3/5/10만원 + 꽝 (DB 연동 확률)
- 가입 보너스: 1복채

## 4. 리텐션 전략

1. **매일 운세**: 카카오톡 알림톡 (오늘의 행운 컬러/시간)
2. **출석 체크**: 매일 방문 유도 보너스
3. **가족 기능**: 구성원 추가 + 가족 궁합
4. **대운(大運) 시스템**: 월별 운세 흐름 추적 (fortune_journal)
