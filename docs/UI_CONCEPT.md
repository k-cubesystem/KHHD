# 해화당 AI 2.0 디자인 컨셉 (UI_CONCEPT)

## 1. 디자인 철학
**"Modern Nobility (현대적 고귀함)"**
전통적인 동양의 신비로움을 덜어내고, KAIST 박사의 정밀한 알고리즘이 느껴지는 데이터 사이언티픽한 유려함을 지향합니다.

## 2. 디자인 키워드
- **Mystic Gold**: 단순한 노란색이 아닌, 금속 질감이 느껴지는 샴페인 골드.
- **Deep Space Black**: 깊이감이 느껴지는 다크 모드 배경.
- **Glassmorphism**: 레이어 간의 구분을 유리 질감으로 처리하여 프리미엄한 깊이감 제공.
- **Dynamic Breathing**: 정적인 UI가 아닌, 기운의 흐름이 느껴지는 미세한 배경 애니메이션.

## 3. 컨셉 제안

### Concept A: "The Golden Fractal" (추천)
- **특징**: 오행(五行)의 상생상극 원리를 기하학적 프랙탈 구조로 시각화.
- **색상**: `Deep Onyx (#0A0A0A)` 베이스 + `Radiant Gold (#D4AF37)` 포인트.
- **UI 요소**: 버튼 클릭 시 기운이 퍼져나가는 듯한 파티클 효과. 분석 리포트는 오래된 비서(秘書)를 현대적으로 재해석한 레이아웃.

### Concept B: "Stellar Logic"
- **특징**: 밤하늘의 성좌와 데이터 노드를 연결한 SF적 감성.
- **색상**: `Midnight Blue (#020617)` 베이스 + `Nebula Silver (#E2E8F0)` 포인트.
- **UI 요소**: 사주 간지를 연결할 때 별자리가 이어지는 듯한 애니메이션. 실시간 데이터 스트리밍 로그를 상단에 노출하여 '공학적' 신뢰도 강조.

## 4. 핵심 UI 컴포넌트 전략
- **분석 대기 화면**: "기운을 조율 중입니다..."라는 문구와 함께 천지인(天地人) 세 개의 구체가 하나로 합쳐지는 3D 애니메이션.
- **결제 전환**: 토스페이먼츠 연동 시, 금빛 실링 왁스가 찍히는 듯한 연출로 '비밀 리포트'라는 가치 전달.
