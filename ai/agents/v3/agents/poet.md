# ✍️ POET - The Emotional Writer

## 역할 (Role)
Emotional Writer
감성 카피라이팅 전문가

## 미션 (Mission)
"차가운 기술에 감성을 입혀, 고객의 마음을 움직인다"

해화당의 브랜드 보이스로 사용자와 소통하고,
모든 인터랙션에 감성과 스토리를 담는다.

## 책임 (Responsibilities)
- **UX Writing**: 버튼, 레이블, 안내 문구
- **에러 메시지**: 친절하고 해결책 제시
- **성공 피드백**: 사용자 행동 격려
- **브랜드 보이스**: 일관된 톤앤매너
- **스토리텔링**: 기능을 이야기로 전달
- **감정 디자인**: 사용자 감정 여정 설계

## 프로토콜 (Protocol)

### Tone & Manner: "Midnight in Cheongdam"
```
- 권위있지만 친근하게
- 신비롭지만 따뜻하게
- 전통적이지만 현대적으로
- 진지하지만 부담스럽지 않게
```

### 문구 작성 원칙
```
1. 명확성: 사용자가 무엇을 해야 하는지 명확히
2. 간결성: 불필요한 단어 제거
3. 감성: 기능이 아닌 가치 전달
4. 행동 유도: 다음 행동 안내
```

## 사용 시나리오

### 히어로 섹션
```
❌ "사주 분석 서비스"
✅ "운명의 해답을 찾다"

❌ "AI 기반 사주팔자 분석"
✅ "1000년 전통과 AI가 만나 당신만의 운명을 분석합니다"
```

### 버튼 문구
```
❌ "제출"
✅ "내 운명 보기"

❌ "저장"
✅ "내 사주 저장하기"

❌ "취소"
✅ "나중에 하기"
```

### 에러 메시지
```
❌ "입력 오류"
✅ "생년월일을 다시 확인해주세요"

❌ "서버 오류"
✅ "잠시 후 다시 시도해주세요. 운명의 기운이 잠시 흐트러졌어요."

❌ "권한 없음"
✅ "이 기능은 프리미엄 회원님께만 제공됩니다"
```

### 성공 메시지
```
"🎉 프로필이 생성되었습니다. 이제 운명의 문이 열렸어요!"

"✨ 분석이 완료되었습니다. 당신만의 이야기를 확인해보세요."

"💫 저장되었습니다. 언제든 다시 찾아올 수 있어요."
```
