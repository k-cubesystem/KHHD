## 📂 구현된 스킬

### 1️⃣ 자동 사주 분석 워크플로우
**파일**: `saju-workflow.skill.json`

생년월일시 입력부터 AI 풀이까지 전체 프로세스를 자동화합니다.

**워크플로우 단계**:
1. 입력 검증 (생년월일시, 성별, 양/음력)
2. 만세력 계산 (lunar-javascript)
3. 오행 균형 분석 (천간/지지 분리)
4. 대운 계산 (10년 주기 타임라인)
5. AI 사주 풀이 (Gemini 2.5 Flash)
6. 리포트 생성

**비용**: 부적 1개 (SAJU_BASIC)
**예상 시간**: < 10초

---

### 2️⃣ 고급 관상 분석 파이프라인
**파일**: `face-analysis-pipeline.skill.json`

얼굴 이미지 업로드부터 개운 이미지 생성까지 전체 파이프라인을 제공합니다.

**파이프라인 단계**:
1. 이미지 업로드 및 전처리 (검증, 최적화)
2. AI 비전 분석 (오관, 삼정, 피부 찰색)
3. 개운 솔루션 생성 (맞춤형 제안)
4. 개운 이미지 생성 (Imagen 3)
5. 통합 리포트 생성

**비용**: 부적 5개 (VISION_ANALYSIS 2 + IMAGE_GENERATION 3)
**예상 시간**: < 15초

**분석 항목**:
- **오관(五官)**: 귀, 눈썹, 눈, 코, 입
- **삼정(三停)**: 상정(이마), 중정(눈~코), 하정(입~턱)
- **피부 찰색**: 기색, 혈색

---

### 3️⃣ 멀티모달 비전 분석 스킬
**파일**: `multimodal-vision.skill.json`

사주 + 관상 + 수상을 통합한 천지인(天地人) 3차원 운명 분석입니다.

**데이터 모달리티**:
- **천(天)**: 사주 - 타고난 운명 (가중치 40%)
- **지(地)**: 관상 - 부모로부터 받은 형체 (가중치 30%)
- **인(人)**: 수상 - 스스로 만드는 운명 (가중치 30%)

**분석 프로세스**:
1. 데이터 수집 (사주, 얼굴 사진, 손금 사진)
2. 병렬 분석 (Promise.all로 동시 처리)
3. 교차 검증 (3가지 결과 일치도 확인)
4. 종합 해석 (통합 리포트 생성)
5. 3D 시각화 (Radar Chart, Timeline)

**비용**: 부적 5개 (PREMIUM_MULTIMODAL) - 패키지 할인 20%
**예상 시간**: < 15초 (병렬 처리)

**특징**:
- 일치도 70% 이상 → 신뢰도 높음
- 불일치 항목 → 변화 가능성 안내
- 개운 로드맵 (즉시/중기/장기)

## 📂 Phase 19 스킬 (2026-01-24 추가)

### 4️⃣ 가족 궁합 매트릭스
**파일**: `family-compatibility-matrix.skill.json`

가족 구성원 간의 사주 기반 궁합을 히트맵으로 시각화합니다.

**핵심 기능**:
- N:N 궁합 매트릭스 (모든 조합 분석)
- 실제 사주 계산 (천간합, 지지합, 지지충)
- 히트맵 시각화 (점수별 색상)
- AI 관계 개선 조언

**궁합 알고리즘**:
- 천간합(갑기/을경/병신/정임/무계): +15점
- 지지합(자축/인해 등): +10점
- 지지충(자오/축미 등): -20점
- 오행 균형: ±10점
- 기본 점수: 70점

**비용**: 기본 무료 / AI 상세 분석 1 부적

### 5️⃣ AI 코칭 스킬 (대화형 운세 상담)
**파일**: `ai-coaching.skill.json`

Gemini와 실시간 대화하며 맞춤형 운세 상담을 받습니다.

**핵심 기능**:
- 실시간 채팅 UI (스트리밍 응답)
- 멀티턴 대화 (이전 대화 기억)
- 음성 입력 지원 (Web Speech API)
- 사주 데이터 자동 로드

**대화 예시**:
- "요즘 직장에서 스트레스가 심한데..."
- "남자친구와 결혼해도 될까요?"
- "이번 달 재물운은 어때요?"

**비용**: 첫 10턴 무료 / 추가 20턴 1 부적

### 6️⃣ 풍수 인테리어 스킬 (3D 시뮬레이션)
**파일**: `fengshui-3d.skill.json`

사주에 맞는 인테리어를 3D로 시뮬레이션합니다.

**핵심 기능**:
- 3D 룸 플래너 (Three.js)
- 사주 기반 길한 방위 표시
- 가구 드래그앤드롭 배치
- 색상 팔레트 추천 (오행 기반)
- AR 미리보기 (선택)

**기술 스택**:
- Three.js + React Three Fiber
- WebXR (AR 모드)

**비용**: 3D 시뮬레이션 2 부적 / AR 추가 1 부적

### 7️⃣ 궁합 바이럴 스킬 (소셜 공유 최적화)
**파일**: `viral-sharing.skill.json`

궁합 결과를 SNS에 공유하여 바이럴 확산을 유도합니다.

**핵심 기능**:
- OG 이미지 자동 생성 (1200x630px)
- SNS별 최적화 메시지 (카톡/인스타/페북)
- 초대 링크 생성 (추천인 추적)
- 공유 리워드 (부적 1개)

**바이럴 루프**:
공유 → 친구 클릭 → 가입 → 또 공유

**목표**: K-factor > 1.0

### 8️⃣ 실시간 웹캠 관상 분석
**파일**: `webcam-face-analysis.skill.json`

웹캠으로 실시간 얼굴 분석 및 가이드를 제공합니다.

**핵심 기능**:
- 실시간 얼굴 감지 (FaceAPI.js)
- 자동 가이드 (각도, 조명, 거리)
- 최적 캡처 타이밍 제안
- 즉시 관상 분석 (오관, 삼정)

**품질 체크**:
- 얼굴 정면: ±15도 이내
- 조명: 100-200 밝기
- 거리: 화면의 20-40%
- 흔들림: < 5px

**비용**: 웹캠 사용 무료 / AI 분석 2 부적

---

### 9️⃣ AR 손금 가이드
**파일**: `ar-palm-guide.skill.json`

AR로 손금을 실시간 분석하고 오버레이로 표시합니다.

**핵심 기능**:
- 실시간 손 감지 (MediaPipe Hands)
- 삼대선 자동 감지 및 AR 오버레이
- 음성 해설 (TTS)
- 양손 비교 모드

**삼대선 분석**:
- 생명선: 건강, 체력
- 지능선: 사고방식, 재능
- 감정선: 애정, 감성

**기술**:
- MediaPipe Hands (21개 랜드마크)
- Three.js (AR 렌더링)
- Web Speech API (음성 해설)

**비용**: AR 세션 무료 / AI 상세 분석 1 부적

## 🚀 사용 방법

### 1. 스킬 파일 읽기
```typescript
import sajuWorkflow from '@/ai/skills/saju-workflow.skill.json';
```

### 2. 워크플로우 실행
```typescript
// 예시: 사주 분석 워크플로우
const result = await executeSajuWorkflow({
  name: "홍길동",
  gender: "male",
  birthDate: "1990-01-15",
  birthTime: "14:30",
  calendarType: "solar"
});
```
