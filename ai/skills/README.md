## 📂 구현된 스킬

### 1️⃣ 자동 사주 분석 워크플로우
**파일**: `saju-workflow.skill.json`

생년월일시 입력부터 AI 풀이까지 전체 프로세스를 자동화합니다.

**워크플로우 단계**:
1. 입력 검증 (생년월일시, 성별, 양/음력)
2. 만세력 계산 (lunar-javascript)
3. 오행 균형 분석 (천간/지지 분리)
4. 대운 계산 (10년 주기 타임라인)
5. AI 사주 풀이 (Gemini 2.5 Flash)
6. 리포트 생성

**비용**: 부적 1개 (SAJU_BASIC)
**예상 시간**: < 10초

---

### 2️⃣ 고급 관상 분석 파이프라인
**파일**: `face-analysis-pipeline.skill.json`

얼굴 이미지 업로드부터 개운 이미지 생성까지 전체 파이프라인을 제공합니다.

**파이프라인 단계**:
1. 이미지 업로드 및 전처리 (검증, 최적화)
2. AI 비전 분석 (오관, 삼정, 피부 찰색)
3. 개운 솔루션 생성 (맞춤형 제안)
4. 개운 이미지 생성 (Imagen 3)
5. 통합 리포트 생성

**비용**: 부적 5개 (VISION_ANALYSIS 2 + IMAGE_GENERATION 3)
**예상 시간**: < 15초

**분석 항목**:
- **오관(五官)**: 귀, 눈썹, 눈, 코, 입
- **삼정(三停)**: 상정(이마), 중정(눈~코), 하정(입~턱)
- **피부 찰색**: 기색, 혈색

---

### 3️⃣ 멀티모달 비전 분석 스킬
**파일**: `multimodal-vision.skill.json`

사주 + 관상 + 수상을 통합한 천지인(天地人) 3차원 운명 분석입니다.

**데이터 모달리티**:
- **천(天)**: 사주 - 타고난 운명 (가중치 40%)
- **지(地)**: 관상 - 부모로부터 받은 형체 (가중치 30%)
- **인(人)**: 수상 - 스스로 만드는 운명 (가중치 30%)

**분석 프로세스**:
1. 데이터 수집 (사주, 얼굴 사진, 손금 사진)
2. 병렬 분석 (Promise.all로 동시 처리)
3. 교차 검증 (3가지 결과 일치도 확인)
4. 종합 해석 (통합 리포트 생성)
5. 3D 시각화 (Radar Chart, Timeline)

**비용**: 부적 5개 (PREMIUM_MULTIMODAL) - 패키지 할인 20%
**예상 시간**: < 15초 (병렬 처리)

**특징**:
- 일치도 70% 이상 → 신뢰도 높음
- 불일치 항목 → 변화 가능성 안내
- 개운 로드맵 (즉시/중기/장기)

## 🔧 기술 스택

### AI 모델
- **Gemini 2.5 Flash Lite**: 사주 분석, 관상 분석
- **Gemini Vision**: 이미지 분석 (얼굴, 손금)
- **Imagen 3**: 개운 이미지 생성

### 라이브러리
- `lunar-javascript`: 만세력 계산
- `@google/generative-ai`: AI 모델 호출
- `recharts`: 데이터 시각화
- `sharp`: 이미지 전처리
- `framer-motion`: 애니메이션

### Server Actions
- `app/actions/ai-saju.ts`
- `app/actions/ai-image.ts`
- `app/actions/analysis-actions.ts`
- `app/actions/wallet-actions.ts`

## 🚀 사용 방법

### 1. 스킬 파일 읽기
```typescript
import sajuWorkflow from '@/ai/skills/saju-workflow.skill.json';
```

### 2. 워크플로우 실행
```typescript
// 예시: 사주 분석 워크플로우
const result = await executeSajuWorkflow({
  name: "홍길동",
  gender: "male",
  birthDate: "1990-01-15",
  birthTime: "14:30",
  calendarType: "solar"
});
```

### 3. 에러 핸들링
각 스킬 파일의 `errorHandling` 섹션 참고
