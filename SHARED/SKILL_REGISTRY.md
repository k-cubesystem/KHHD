# 스킬 레지스트리

> 재사용 가능한 스킬(작업 패턴/함수/프롬프트)을 등록합니다.
> 자세한 구현은 `TEAM_F_SKILLS/registry/` 폴더를 참조하세요.

---

## 등록 형식

```
### SKILL-{번호}: {스킬명}

- **설명**: 무엇을 하는 스킬인가
- **입력**: 필요한 파라미터
- **출력**: 반환값 또는 결과물
- **파일**: 구현 위치
- **등록일**: YYYY-MM-DD
```
