**CHANI -- 가장 리텐션 높은 점성술 앱**

- 최초의 실시간 전문가 상담 기능

## 2. "과거 맞추기 -> 현재 공감 -> 미래 예측" 3단계 신뢰 구축법

### 2.1 심리학적 근거

**Barnum Effect (바넘 효과)**
- 일반적인 성격 묘사를 자신에게만 해당한다고 느끼는 인지 편향
- 가장 효과적인 구문: "at times" 패턴 ("때로는 ~하지만 때로는 ~합니다")
- 긍정적 서술 위주 + 약간의 부정적 요소 = 신뢰도 극대화

**Cold Reading (콜드 리딩)**
- 사전 정보 없이 상대의 외모, 나이, 행동, 직업에서 단서를 추출하는 기법
- 고확률 추측 후 적중하면 강조, 빗나가면 빠르게 넘기기
- "자신감 있는 전달" + "심볼(크리스탈볼, 타로 등) 활용" = 권위 생성

### 2.2 해화당 AI에서의 3단계 구현 설계

**[1단계] 과거 맞추기 -- "소름 돋는 첫인상"**

사주 데이터로 과거를 역추산하는 구체적 방법:

(a) 대운(10년 주기) 역추산
- 사용자의 과거 대운을 계산하여 해당 시기의 특성을 서술
- 예: 30대 중반 사용자 -> "20대 중반에 큰 변화가 있었을 것입니다. 직업이나 거처가 바뀌었거나, 인생의 방향을 바꾸는 결정을 하셨을 겁니다"
- 근거: 대운 교체기(대운이 바뀌는 시점)는 인생의 전환점과 높은 상관관계

(b) 세운 충/합 역추산
- 과거 세운(연운)에서 원국과 충(衝)이 발생한 해 = 큰 사건이 일어난 해
- 예: "2020년(경자년)에 건강이나 가정에서 예상치 못한 사건이 있었을 가능성이 높습니다"
- 일지 충 = 배우자/가정 관련 사건
- 월지 충 = 직장/사회적 변화
- 년지 충 = 부모/고향 관련 변화

(c) 격국/용신 기반 성격 역추산
- "어릴 때부터 사람들 사이에서 눈치가 빠르셨을 겁니다" (편인격)
- "학창시절 규칙에 잘 안 맞았을 가능성" (상관격)
- "가정에서 책임감을 일찍 지셨을 겁니다" (정관격, 인성약)

(d) 물상론 연결
- 갑목 일간: "소나무처럼 곧은 성격, 꺾일지언정 타협하지 않는 성향"
- 계수 일간: "주변 사람들의 감정을 잘 읽어내는 능력"

**구현 방법 (프롬프트 레벨):**
```
[과거 역추산 섹션]
아래 과거 대운/세운 데이터를 바탕으로, 내담자가 "이걸 어떻게 알지?"라고
느낄 만큼 구체적인 과거 사건을 3개 이상 역추산하세요.
- 시기를 명확히 지정 ("20대 중반", "2019~2020년경")
- 사건의 카테고리를 특정 ("연애/이별", "이직/퇴사", "이사", "건강 문제")
- 감정 상태까지 묘사 ("당시 상당히 혼란스러웠을 겁니다")
- 단, 100% 확정형이 아닌 "~했을 가능성이 높습니다" 형태로 서술
```

**[2단계] 현재 공감 -- "지금 내 마음을 읽는다"**

(a) 현재 세운/월운 분석
- 2026년 병오년의 기운이 사용자 원국과 만나는 지점을 분석
- "지금 이 시기, 마음속에 새로운 것을 시작하고 싶은 충동이 있으실 겁니다" (병화 기운)

(b) 프로필 컨텍스트 활용 (해화당 기구현)
- 직업, 결혼 상태, 관심사 정보를 사주 해석과 결합
- "IT 업계에서 일하고 계신다면, 올해 특히 기술적 전환점에 서 있는 느낌이실 겁니다"

(c) 계절/절기 컨텍스트
- 현재 절기가 사주에 미치는 영향 구체화
- "지금 경칩을 지난 시점, 겨울 동안 억눌렸던 에너지가 움직이기 시작하는 때입니다"

**구현 방법 (프롬프트 레벨):**
```
[현재 공감 섹션]
현재 세운/월운과 원국의 상호작용을 분석하여, 내담자가 "바로 지금 느끼고 있는 것"을
맞추세요. 특히:
- 현재의 감정 상태 ("요즘 결단을 내려야 할 것 같은 압박감")
- 현재의 고민 카테고리 ("재정적 불안" 또는 "관계의 변화")
- 현재의 에너지 레벨 ("체력적으로 좀 지치신 느낌")
내담자 프로필의 직업/관심사와 연결하여 더 구체적으로 서술하세요.
```

**[3단계] 미래 예측 -- "구체적 시기 + 행동 가이드"**

(a) 시기 특정
- 대운 교체 시점, 세운 길신이 오는 해, 월운에서 용신이 강해지는 달
- "2026년 7~8월이 올해 가장 좋은 기회의 창입니다"
- "2027년 하반기에 새로운 대운이 시작되면서 인생의 큰 전환이 옵니다"

(b) 행동 지침의 구체성
- 추상적: "좋은 일이 있을 것입니다" (X)
- 구체적: "7월 셋째 주에 중요한 미팅이나 계약이 있다면, 적극적으로 추진하세요. 이 시기에 시작한 일은 3개월 내에 결실을 봅니다" (O)

(c) 개운법의 실용성
- 추상적: "동쪽이 좋습니다" (X)
- 구체적: "아침 출근길에 한강공원 동쪽 산책로를 10분만 걸어보세요. 갑목(甲木)인 당신에게 수(水)의 기운이 올해 최고의 에너지원입니다" (O)

### 4.2 궁합

**사용자가 가장 궁금한 것 (우선순위 순):**
1. "우리 잘 맞아?" (점수) -- 가장 먼저 확인
2. "어떤 점이 안 맞아?" (갈등 포인트) -- 가장 오래 읽는 부분
3. "어떻게 하면 잘 살 수 있어?" (해결책) -- 가장 가치 있는 부분

**풀이 구조:**
```
[1] 궁합 점수 (즉시 만족)
  100점 만점 총점 + 8개 카테고리별 점수 (레이더 차트)
  - 성격 궁합, 연애 궁합, 성적 궁합, 재물 궁합
  - 소통 궁합, 가치관 궁합, 가정 궁합, 운명 궁합

[2] 두 사람의 첫인상 (과거 맞추기)
  "처음 만났을 때 ~한 느낌이었을 것입니다"
  "서로에게 끌린 이유는 ~입니다" (오행 상생/상극 기반)

[3] 꿀 궁합 포인트 (좋은 점 먼저)
  두 사람이 가장 잘 맞는 3가지 영역
  구체적 상황 예시와 함께 ("여행할 때 특히 잘 맞습니다")

[4] 갈등 지뢰밭 (솔직한 분석) ★ 가장 많이 읽는 부분
  부딪힐 수밖에 없는 지점 3가지
  각 갈등에 대한 구체적 시나리오
  "금전 관련 결정에서 충돌이 예상됩니다. A는 ~하려 하고 B는 ~하려 합니다"

[5] 관계 매뉴얼 (해결책)
  갈등별 구체적 대화법, 행동 지침
  "재정 갈등 해결: 매월 1일 함께 가계부를 정리하는 루틴을 만드세요"

[6] 타이밍 가이드
  관계에서 조심해야 할 시기 / 가장 좋은 시기
  "2026년 하반기에 함께 여행하면 관계가 깊어집니다"

[7] 유명인 궁합 비교
  "두 분의 궁합은 [유명인 커플]과 비슷한 패턴입니다"
  친근하고 공유하고 싶은 콘텐츠 생성
```

**바이럴 포인트:** 궁합 결과는 반드시 두 사람이 함께 보게 설계. "상대방에게 공유하기" CTA를 점수 옆에 배치.

### 4.3 관상

**사용자가 가장 궁금한 것:**
1. "나는 부자상? 복상?" -- 운명적 궁금증
2. "내 얼굴에서 고쳐야 할 점?" -- 실용적 관심
3. "관상으로 본 성격" -- 자기 이해 욕구

**풀이 구조:**
```
[1] 관상 총점 + 한 줄 요약
  "당신은 [귀한 인물상/사업가상/예술가상]의 관상입니다"
  + 유명인 비유 ("~와 비슷한 기질의 관상")

[2] 부위별 6단계 분석 (기구현)
  이마(초년운) -> 눈(중년운) -> 코(재물운) -> 입(말년운) -> 귀(선천운) -> 턱(의지력)
  각 부위별 점수 + 한 줄 해석
  시각적 하이라이트 (사진 위에 부위별 점수 오버레이)

[3] 관상 + 사주 교차 분석 ★ 해화당만의 차별점
  "관상으로 보면 재물운이 강한데, 사주에서도 편재가 강합니다 -> 확실한 사업가 기질"
  "관상에서 눈이 예리한데, 사주의 현침살과 일치합니다 -> 분석/연구 분야 적합"

[4] 관상 개운법
  메이크업/안경/수염 등 외모 조정으로 운을 바꾸는 실용적 조언
  "눈썹을 조금 더 진하게 그리면 결단력의 기운이 강해집니다"

[5] 나이별 운세 흐름
  관상의 부위와 나이대 매핑 (이마=15~30세, 눈=31~50세, 코/입=51세~)
```

## 5. 리텐션 전략

### 5.1 일간 루프 (DAU 극대화)

**오늘의 운세 푸시 알림 설계:**

톤: Co-Star식 날카로운 한 줄 + 해화당식 따뜻한 조언

예시:
```
[아침 알림]
"오늘 편재의 기운이 강합니다.
충동구매 조심. 대신 아끼던 아이디어를 실행하기엔 최적의 날."

[저녁 알림 (선택적)]
"오늘 하루, 예상대로 흘러갔나요?
내일은 인성의 기운. 배움과 성찰의 시간입니다."
```

**행운의 숫자/색상/방위:** 매일 변경되는 간단한 게임화 요소

### 5.2 주간/월간 루프

**월초 알림:** "3월 운세가 도착했습니다. 이달 가장 조심할 날은 3월 17일."
**시기별 경고:** "이번 주 수요일은 구설수 주의. 중요한 발언은 목요일로 미루세요."
**절기 알림:** "오늘은 춘분(春分). 새로운 시작의 에너지가 가장 강한 날입니다."

### 5.3 이벤트 루프

**24절기 이벤트 (기구현: seasonal-events.ts)**
- 각 절기마다 특별 운세 콘텐츠 무료 해금
- "입추 특별 운세: 하반기 전환점의 비밀"

**생일 운세:** 사용자 생일 전 7일부터 카운트다운 -> 생일 당일 특별 분석 무료 제공
**기념일 알림:** 가족/연인 생일 전 궁합 리포트 업데이트 알림

### 5.4 관계 기반 루프 (바이럴 + 리텐션)

**궁합 업데이트:**
- 등록된 가족/연인의 세운이 바뀔 때마다 궁합 리포트 자동 업데이트
- "이번 달 [파트너 이름]과의 궁합이 변화했습니다"

**가족 운세:**
- 가족 구성원 전체의 월간 운세 요약 대시보드
- "이번 달 가족 중 [자녀 이름]에게 특별한 기운이 있습니다"

### 5.5 적중률 피드백 시스템 ★ 혁신 기능

```
[운세 복기 시스템]
매주 금요일: "이번 주 운세, 맞았나요?"
- 적중/비적중 피드백 수집
- 적중률 퍼센트 표시 ("당신의 운세 적중률: 73%")
- 적중 사례 하이라이트 ("지난 달 재물운 경고가 맞았습니다!")

[효과]
1. 맞으면: "역시 진짜다" -> 신뢰 강화 -> 유료 전환
2. 안맞으면: "다음에는 맞을까?" -> 호기심 -> 재방문
3. 데이터 수집: 사용자 피드백으로 프롬프트 지속 개선
```

## 6. 풀이 톤 & 스타일

### 6.1 해화당 톤 매트릭스

**기본 원칙: "솔직하되 희망적" (Honest but Hopeful)**

사주에 좋은 것이 있을 때:
- (X) "대박입니다! 엄청난 운이 들어옵니다!"
- (O) "이 시기에 시작하는 일은 좋은 결실을 맺을 가능성이 높습니다. 다만 과욕은 금물."

사주에 안좋은 것이 있을 때:
- (X) "큰일입니다. 이 시기에는 아무것도 하지 마세요."
- (O) "이 시기는 준비와 정비의 시간입니다. 나무도 겨울에 뿌리를 내립니다. 구체적으로 ~하면 오히려 전화위복의 기회가 됩니다."

### 6.2 톤 스펙트럼

```
[해화당 톤 = 좌표 (3, 4)]

전문적 5 |
         |     CHANI
         |           ★ 해화당
격조있는 3 |  포스텔러
         |
가벼운  1 |  점신    Co-Star
         +---+---+---+---+---
          1   2   3   4   5
       부정적            긍정적
```

해화당의 톤 포지션:
- Co-Star보다 따뜻하지만, 점신보다 격조 있게
- 포스텔러보다 감성적이지만, CHANI보다 전문적

### 6.3 구체적 톤 가이드라인

## 7. 해화당 차별화 전략 종합

### 7.1 "사주 + X" 교차 분석 (타서비스에 없는 해화당만의 무기)

해화당이 가진 가장 큰 차별점: 사주 + 관상 + 풍수를 하나의 시스템에서 교차 분석.

```
[천지인 통합 분석 = 사주(천) + 관상/손금(지) + 의지(인)]

"사주에서 재물운이 강한데 관상에서도 코가 풍만합니다 -> 확신을 갖고 투자하셔도 됩니다"
"사주에서 관성이 강한데 관상에서 이마가 넓습니다 -> 조직에서의 승진 가능성이 매우 높습니다"
"사주의 용신이 수(水)인데, 현재 집의 북쪽이 막혀 있습니다 -> 풍수 조정이 필요합니다"
```

이 교차 분석은 단일 서비스(사주만, 관상만, 풍수만)에서는 불가능하며, 해화당의 핵심 경쟁 우위.

### 7.2 "AI + 전통 명리" 하이브리드

- 점신: AI가 약하고 상담사 연결에 의존
- 포스텔러: AI가 강하지만 전통 명리 깊이가 약함
- 사주나루: 전통(신점)이 강하지만 AI가 약함
- 해화당: haehwajigi.md의 명리요강/사주첩경/자평진전 3대 비법서 + Gemini AI = 양쪽 모두의 장점

### 7.3 바이럴 콘텐츠 설계 (VIRAL 발동)

**공유하고 싶은 결과 포맷:**
1. 사주 유형 카드: "당신은 [물상론 기반 유형명]입니다" (예: "바위틈의 난초", "깊은 바다의 진주")
2. 궁합 점수 카드: 두 사람의 점수 + 한 줄 코멘트 (SNS 공유용 OG 이미지)
3. 관상 분석 카드: 얼굴 사진 위에 부위별 점수 오버레이
4. 오늘의 운세 카드: 날카로운 한 줄 + 행운 아이템 (Co-Star 밈 전략 한국화)

**토스 전생 프로필 이벤트 학습:**
- "결과가 사람마다 다르다는 것을 발견하는 재미"가 공유의 핵심
- "너는 뭐 나왔어?" 질문을 유도하는 유니크한 결과 유형 설계
- 물상론 기반 유형 분류: 10천간 x 12지지 = 120가지 유니크 유형
