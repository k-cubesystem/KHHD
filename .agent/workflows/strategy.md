description: [Workflow] 단계별 상세 액션 플랜
---

#### **Step 1: 명령 분석 및 1차 보고 (Strategy)**
사용자가 기획을 던지면 **Gemini 3 Pro**는 다음을 수행합니다.
1.  명령의 기술적 타당성 조사.
2.  `/docs/STRATEGY.md` 업데이트: "이러한 방향으로 개발하겠습니다. 승인하시겠습니까?"
3.  `/docs/UI_CONCEPT.md` 업데이트: "디자인 컨셉 A(미니멀), B(고전 명리), C(퓨처리즘) 중 선택해 주세요."

#### **Step 2: 자율주행 모드 돌입 (Execution)**
승인 후, 인간의 개입 없이 진행됩니다.
1.  **임무 쪼개기:** 전체 작업을 10~20개의 작은 태스크로 분해.
2.  **AI 배치:** 
    - 사주 알고리즘 검증 -> Gemini 3 Pro.
    - UI 컴포넌트 코딩 -> Claude.
    - 단순 유틸리티 및 반복 작업 -> Flash 계열.
3.  **에러 루프:** 에러 발생 시 스스로 로그를 분석하고, 해결 안 될 경우 즉시 상위 모델(Claude)에게 디버깅 임무 하달.

#### **Step 3: 중간 점검 및 최종 보고 (Report)**
1.  작업 완료 후 통합 테스트 수행.
2.  `/docs/MISSION_LOG.md`를 최종 업데이트하여 사용자에게 전체 작업 완료 보고.
3.  다음 확장 가능한 기능 제안.
