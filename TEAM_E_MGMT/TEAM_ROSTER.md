# 팀 로스터

> 해화당 멀티 에이전트 시스템 v6.0

## 에이전트 목록

| ID     | 역할          | 책임 영역                      | 주요 경로                     |
| ------ | ------------- | ------------------------------ | ----------------------------- |
| ARCH   | 아키텍처      | 설계, 의사결정, 기술 부채 관리 | `MEMORY/`, `SHARED/`          |
| FEAT   | 기능 개발     | 신규 기능 구현, 서버 액션      | `app/actions/`, `components/` |
| FIX    | 버그 수정     | 이슈 분석 및 수정, 핫픽스      | 전체                          |
| TEST   | 테스트        | E2E, 유닛 테스트 작성          | `e2e/`, `__tests__/`          |
| REVIEW | 코드 리뷰     | PR 리뷰, 컨벤션 준수 확인      | 전체                          |
| SKILL  | 스킬 관리     | 재사용 패턴 발굴 및 등록       | `TEAM_F_SKILLS/`              |
| MGMT   | 프로젝트 관리 | 스프린트 조율, 로그 관리       | `TEAM_E_MGMT/`                |

## 에이전트 간 협업 원칙

1. 작업 시작 전 `SHARED/TEAM_STATUS.md` 확인
2. 의사결정은 `MEMORY/DECISION_LOG.md`에 기록
3. 재사용 패턴 발견 시 `SHARED/SKILL_REGISTRY.md`에 등록
4. 컨벤션 위반 시 REVIEW 에이전트에 에스컬레이션
